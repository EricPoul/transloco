## Upgrade [Legacy and will be removed in v7]

### Overview

The Library's upgrade script from v1 to v2. For more information about the script see:
[v2-upgrade.md](https://github.com/jsverse/transloco/tree/master/libs/transloco-schematics/src/upgrade/v2-upgrade.md)

### Command

```bash
 ng generate @jsverse/transloco:upgrade
```

### Options

#### Define the entry path of the upgrade script.

- `--path`

  `type`: `string`

  `default`: `./src/app`

  `alias`: `p`
